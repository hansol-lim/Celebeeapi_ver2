## Create Virtual Environment
---
```
# 가상환경 생성
$ python3 -m venv venv

# 가상환경 실행
$ source venv/bin/activate

# 관련 라이브러리 설치
(venv) $ pip install -r requirements.txt
```

## Execute Flask App
---
```
# .env 설정 없을 시
$ export FLASK_APP=beamme.py
$ export FLASK_ENV=development

# flask 실행
(venv) $ flask run
```
